## 📲 スマートフォンからのアクセス方法

### 方法1: QRコードをスキャン
![QRコード](https://api.qrserver.com/v1/create-qr-code/?size=300x300&data=https://unarticulative-adella-unprostrated.ngrok-free.dev)

1. スマートフォンのカメラアプリを起動
2. 上記のQRコードをスキャン
3. 表示されたリンクをタップ

### 方法2: URLを直接入力
1. スマートフォンのブラウザを開く
2. 上記のURLをコピーして入力
3. アクセス

### 方法3: 短縮URLを使用
長いURLが入力しづらい場合は、bit.lyなどで短縮できます。

## 🚀 使い方

### 1. アプリにアクセス
上記のいずれかの方法でアプリにアクセスします。

### 2. ルームに参加
- **新規ルーム作成**: 「Generate Room ID」をタップ
- **既存ルームに参加**: ルームIDを入力

### 3. 通話開始
「Join」ボタンをタップして通話を開始します。

### 4. 機能の利用
- 🎥 **カメラ**: カメラのON/OFF切り替え
- 🎤 **マイク**: ミュートの切り替え
- 💬 **チャット**: テキストメッセージの送信
- 📱 **画面共有**: 画面を共有（PCのみ）
- 🚪 **退出**: 「Leave」で通話から退出

## ⚙️ トラブルシューティング

### カメラ・マイクが使えない場合
1. ブラウザの設定でカメラ・マイクの許可を確認
2. デバイスの設定 > アプリ > ブラウザ > 権限を確認

### 接続できない場合
1. Wi-Fiまたはモバイルデータ通信を確認
2. ブラウザのキャッシュをクリア
3. 別のブラウザで試す

### 音声・映像が途切れる場合
- 安定したネットワーク環境で使用
- 他のアプリを終了してメモリを確保

## 📊 パフォーマンス推奨設定

### スマートフォンでの最適設定
- **ビデオ解像度**: 480p（データ節約）
- **フレームレート**: 15-30fps
- **音声**: ノイズキャンセリング有効

### ネットワーク要件
- **最小**: 1Mbps（音声のみ）
- **推奨**: 3Mbps（ビデオ+音声）
- **最適**: 5Mbps以上（高画質）

## 🔒 セキュリティ

- すべての通信はHTTPSで暗号化
- ルームIDは推測困難なランダム文字列
- P2P接続で低遅延を実現

## ⚠️ 注意事項

- ngrok無料版は8時間で自動切断されます
- URLは起動のたびに変更されます
- 同時接続数は使用するサーバーに依存します
